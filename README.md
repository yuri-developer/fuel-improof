## Софт на FueL

Telegram: **@improof**

### Установка

`npm install`

Вписываем приватники в файл `private_keys.txt`

Запуск `npm start`

Настройки `config.ts`

### Возможности

* Создание новых кошельков
* Получение токенов из крана
* Отправка транзакций на свой кошелек
* Отправка транзакций на случайный кошелек
